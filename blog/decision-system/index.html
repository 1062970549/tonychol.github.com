Decision System

决策支持系统

第七章

大纲

  1. 概况
  2. 效用理论
  3. 多目标决策（运筹学）
  4. 期望损益决策

概况

  1. 定义
    1. 在有风险或者不确定性的条件下指定决策，重要：（定量！）。
    2. 运筹学的一支
    3. 控制论的延伸
    4. 管理科学的组成部分
    5. 是社会科学和自然科学的交叉，典型的软科学。
  2. 分类
    1. 单人决策/多人决策
    2. 单目标/多目标决策
    3. 单步决策/多步决策
    4. 确定型决策 风险型决策 etc
  3. 特点
    1. 面临选择
    2. 后果的不确定性
    3. 后果的价值待定
    4. 例子： 火灾保险 库存问题 （腐烂问题、报贩进货）
  4. 几点说明
    1. 经济人的假设
    2. 好的决策 = 好的结果？？？ 不一定的。好的决策是逻辑上的，好的结果是愿望（例子：炒股票）
  5. 随机决策问题的特点：
    1. 自然状态不确定 ——概率表示
    2. 后果价值待定 ——以效用度量

效用理论

  1. 效用的定义
    1. 严格序 a is preferred to b
    2. 无差异
    3. 弱序
  2. 展望/预期
    1. 定义
      • 各种后果极其概率的组合。
    2. 确定当量
    3. 效用函数 u(P1) u(P2)。。。
      • P1 P2 为后果
    4. 效用存在性公理（理性行为公理）
      • 即便是后果再差，效用也不会是负无穷大，效用是有界的。
      • 效用函数的基数性和序数性
        • 基数效用：在展望集P 上（考虑后果极其概率分布），是实数。
        • 序数效用：直接就是正整数， 1,2,3,4……
      • 之后效用直接用货币来表示。
        • Friedmann-Savage 货币效用曲线（上下有界）
        • 决策树
